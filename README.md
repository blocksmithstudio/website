# Blocksmith Studio Website

Minimal modern one-page website for Blocksmith Studio, built with Next.js App Router.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Hero video

Replace this file path with your Minecraft animation video:

```txt
public/videos/blocksmith-hero.mp4
```

The landing page already uses that path in the hero section.
